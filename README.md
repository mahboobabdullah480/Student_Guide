# 🚀 دليل طالب تقنية المعلومات (IT Student Guide)

**منصة متكاملة تجمع بين التخصصات، الدورات، المهارات، الأدوات، المشاريع، ونصائح النجاح الأكاديمي والمهني.**

موقع ويب تفاعلي ومطور خصيصاً لمساعدة طلاب تخصصات تكنولوجيا المعلومات (IT) في تحديد مساراتهم الأكاديمية والمهنية، واكتساب المهارات المطلوبة لسوق العمل بناءً على خطط مدروسة ومشاريع حقيقية.

---

### 🌟 مميزات المنصة (Features)
* 🎯 **اختيار المجال والتخصص:** توجيه الطلاب نحو التخصص الأنسب لقدراتهم وطموحاتهم.
* 📚 **الدورات والمهارات:** دليل شامل لأهم المهارات التقنية والكورسات التعليمية الموصى بها.
* 🛠️ **الأدوات والمراجع:** توفير المصادر والأدوات التقنية التي يحتاجها كل مطور ومبرمج.
* 💻 **المشاريع العملية:** أفكار ومسارات لبناء مشاريع حقيقية تؤهل الطلاب لسوق العمل.
* 🌓 **الوضع الداكن (Dark Mode):** واجهة مستخدم مريحة تدعم التبديل السلس بين الوضعين المضيء والداكن.
* 📱 **تصميم متجاوب بالكامل (Fully Responsive):** متوافق مع جميع الشاشات والأجهزة (الهواتف، الأجهزة اللوحية، والحواسيب).

---

### 🛠️ التقنيات المستخدمة (Tech Stack)
* **Frontend:** HTML5, CSS3, JavaScript (Vanilla JS), Font Awesome Icons
* **Backend:** PHP
* **Architecture:** Responsive Grid System, Jumbotron Layout
